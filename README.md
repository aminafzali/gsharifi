# پت وت - پلتفرم دامپزشکی آنلاین

طراحی مدرن و تمیز برای پلتفرم دامپزشکی آنلاین **بدون استفاده از سایه**.

## 🎨 ویژگی‌های طراحی جدید

### ✅ سکشن‌های اضافه شده:

1. **سکشن مشاورین** - معرفی دامپزشک‌ها با عکس، امتیاز و تخصص
2. **سکشن حیوانات** - SVG های بزرگ 100x100 با انیمیشن‌های پیشرفته
   - 🐕 سگ: دم تکان می‌خورد، گوش‌ها حرکت می‌کنند، پنجه‌ها می‌پرند
   - 🐈 گربه: دم می‌چرخد، سبیل‌ها می‌لرزند، چشم‌ها پلک می‌زنند
   - 🐦 پرنده: بال‌ها بالا پایین می‌شوند، سر تکان می‌خورد، تاج حرکت می‌کند
   - 🐹 جونده: گوش‌ها می‌لرزند، سبیل‌ها می‌لرزند، بینی بو می‌کشد
   - 🐠 ماهی: شنا می‌کند، دم و باله‌ها حرکت می‌کنند، حباب‌ها بالا می‌روند
3. **سکشن مقالات** - ۳ مقاله آموزشی با تصویر و توضیحات
4. **سکشن مزایا** - با آیکون‌های خطی SVG
5. **سکشن نظرات کاربران** - ۳ نظر واقعی با امتیاز ستاره‌ای
6. **بنر تخفیف ۵۰٪** - بنر قرمز جذاب

### 🎯 تغییرات عمده:

- ❌ **بدون سایه**: تمام shadow ها حذف شده
- 🟤 **هدر و فوتر قهوه‌ای**: رنگ `#C4A57B`
- 🔵 **لوگو ساده**: طراحی خطی شبیه به لوگوی طرح اصلی

## 🎭 انیمیشن‌های SVG حیوانات

### طراحی دقیق و جزئیات:

هر حیوان با **SVG خطی 100x100 پیکسل** طراحی شده با جزئیات کامل:

#### 🐕 سگ (Dog):
- **بدنه**: دایره اصلی با paw print
- **گوش‌ها**: بیضی چرخیده که با hover تکان می‌خورند
- **پنجه‌ها**: دایره‌های کوچک که بالا پایین می‌پرند
- **دم**: خم منحنی که چپ و راست می‌چرخد
- **چشم‌ها**: نقاط سیاه
- انیمیشن: `dogTailWag` - دم هر 0.8 ثانیه تکان می‌خورد

#### 🐈 گربه (Cat):
- **بدنه**: بیضی
- **گوش‌ها**: مثلثی تیز که می‌لرزند
- **سبیل‌ها**: 4 خط از هر طرف که با hover حرکت می‌کنند
- **دم**: منحنی بلند که می‌چرخد
- **چشم‌ها**: بیضی که پلک می‌زنند
- انیمیشن: `catWhiskerTwitch` + `catEyeBlink` - سبیل‌ها می‌لرزند و چشم‌ها پلک می‌زنند

#### 🐦 پرنده (Bird):
- **بدنه**: بیضی عمودی
- **سر**: دایره با منقار نارنجی
- **بال‌ها**: خطوط منحنی که بالا پایین می‌شوند
- **پاها**: خطوط نازک نارنجی
- **تاج**: سه خط روی سر
- **دم**: 3 خط از پایین
- انیمیشن: `birdWingFlap` - بال‌ها هر 0.4 ثانیه بال می‌زنند

#### 🐹 جونده (Rodent):
- **گوش‌های بزرگ**: بیضی‌های بزرگ که می‌لرزند
- **سبیل‌ها**: 6 خط (3 از هر طرف) که vibrate می‌شوند
- **بینی صورتی**: دایره که بزرگ و کوچک می‌شود (بو می‌کشد)
- **دندان‌ها**: 2 خط سفید
- **دم**: خم بلند
- انیمیشن: `rodentNoseSniff` + `rodentBounce` - کل بدن می‌پرد

#### 🐠 ماهی (Fish):
- **بدنه**: بیضی افقی با فلس‌ها
- **دم**: 3 خط که تکان می‌خورد
- **باله‌های بالا/پایین**: که موج می‌زنند
- **چشم**: دایره با نقطه سفید درون
- **حباب‌ها**: 3 دایره که بالا می‌روند
- انیمیشن: `fishSwim` + `bubbleFloat` - ماهی شنا می‌کند و حباب‌ها بالا می‌روند

### ویژگی‌های انیمیشن:

- ✅ **Hover-based**: انیمیشن‌ها فقط با hover فعال می‌شوند
- ✅ **Smooth**: استفاده از `ease-in-out` برای نرمی
- ✅ **Performance**: بهینه برای موبایل
- ✅ **Accessibility**: `prefers-reduced-motion` پشتیبانی می‌شود
- ✅ **Breathing**: همه SVG ها یک انیمیشن تنفس ملایم دارند
