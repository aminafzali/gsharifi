برای استفاده آفلاین، دو فایل three.min.js و tailwind.js را در این پوشه قرار دهید.

روش ۱ - PowerShell:
   cd js
   powershell -ExecutionPolicy Bypass -File download.ps1

روش ۲ - Node.js (در صورت نصب بودن):
   cd js
   node download-node.js

روش ۳ - دانلود دستی:
   1. three.min.js: https://cdn.jsdelivr.net/npm/three@0.160.0/build/three.min.js
   2. tailwind.js: https://cdn.tailwindcss.com
   با مرورگر لینک را باز کنید، Ctrl+S بزنید و در این پوشه ذخیره کنید.
